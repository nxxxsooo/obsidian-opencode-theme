# OpenCode Theme for Obsidian

A dark theme for [Obsidian](https://obsidian.md) inspired by the [OpenCode](https://github.com/sst/opencode) terminal aesthetic.

## Features

- **High contrast** dark color scheme optimized for readability
- **Terminal-inspired** aesthetics matching OpenCode/Ghostty
- **Font smoothing** with antialiasing and text thickening
- **Accent color support** - customize via Obsidian settings
- **Responsive font scaling** using rem units

## Installation

### From Obsidian Community Themes

1. Open Obsidian Settings
2. Go to **Appearance** → **Themes**
3. Click **Manage** → **Browse**
4. Search for "OpenCode"
5. Click **Install and use**

### Manual Installation

1. Download `theme.css` and `manifest.json`
2. Create folder: `{vault}/.obsidian/themes/OpenCode/`
3. Place both files in the folder
4. Enable in Obsidian: **Settings** → **Appearance** → **Themes**

## Recommended Settings

**Fonts:**
- Interface: PingFang SC / SF Pro Text
- Text: PingFang SC
- Monospace: JetBrains Mono

**Accent Color:** `#8a6cc4` (default purple, customizable)
